О проекте: Mini Task Board

Назначение: Веб-приложение для управления задачами с использованием REST API архитектуры.
Backend: Python / Flask / SQLAlchemy (ORM). Реализованы CRUD-запросы для работы с данными.
Database: SQLite. Обеспечивает надежное хранение и быстрый доступ к информации.
Frontend: Адаптивная верстка (HTML5/CSS3) и динамическое обновление данных через JavaScript (Fetch API).
Version Control: Проект разработан с применением системы Git для совместной работы.

Проект демонстрирует навыки фуллстек-разработки и проектирования баз данных.
