API documentation

Task endpoints are JSON. You must be logged in (session cookie). From the board page, the browser sends the cookie automatically.

  • GET /tasks — list your tasks
  • POST /tasks — body {"title": "..."} — create (starts in todo)
  • PATCH /tasks/{id} — body {"status": "todo"|"doing"|"done"} and/or {"title": "..."} — move or rename
  • DELETE /tasks/{id} — delete

Open Swagger UI (try requests with your session; log in first in another tab).
Raw spec: /openapi.json
